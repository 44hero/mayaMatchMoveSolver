mmSolver.tools.selection¶

This package contains all selection tools related to solver, marker, and bundle.

Tools¶

Common tools for manipulating selection, specific to mmSolver.

mmSolver.tools.selection.tools.swap_between_selected_markers_and_bundles()¶

Toggles the selection of Markers and Bundles.

If a marker is selected, the attached bundle will be selected and vice versa.

mmSolver.tools.selection.tools.select_both_markers_and_bundles()¶

Get the connected Markers and bundles, and select them.

Convert¶

The mmSolver.tools.selection.convert module provides functions to convert between various node types. Currently the following convertions are supported:

  • Markers -> Bundles
  • Bundles -> Markers
  • Markers -> Cameras
mmSolver.tools.selection.convert.get_bundles_from_markers(nodes)¶

Convert Marker nodes into Bundle nodes.

Parameters:nodes (list of str) – Maya nodes to convert into Bundles (expected to be Marker nodes, but other node types will not cause errors).
Returns:All Maya nodes connected to Marker nodes as Bundles.
Return type:list of str

Filter Nodes¶

Helpful tools to filter a list of nodes into pre-defined categories.

mmSolver.tools.selection.filternodes.get_nodes(nodes)¶

Return nodes sorted into mmSolver-specific object type categories.

Supported categories are:

  • ‘camera’
  • ‘marker’
  • ‘markergroup’
  • ‘bundle’
  • ‘attribute’
  • ‘collection’
  • ‘other’
Parameters:nodes (list of str) – Maya nodes to categorise.
Returns:Dictionary with lists for each object type.
Return type:dict
