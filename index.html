mmSolver - Maya MatchMove Solver¶

Welcome to mmSolver! mmSolver is a Bundle Adjustment solver for Autodesk Maya, for solving MatchMove tasks by adjusting Maya attributes.

This documentation is for the GitHub project, and details the tools, solver design, python scripting, plug-in command and node details.

If you are new to mmSolver, take a look at the video Tutorials.
