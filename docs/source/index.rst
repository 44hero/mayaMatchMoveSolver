
mmSolver - Maya MatchMove Solver
================================

Welcome to `mmSolver`! `mmSolver` is a Bundle Adjustment solver for
Autodesk Maya, for solving MatchMove tasks by adjusting Maya
attributes.

This documentation is for the GitHub project_, and details the tools,
solver design, python scripting, plug-in command and node details. For
installation instructions see the *INSTALL.md* file with mmSolver, or
the latest install development instructions_.

If you are new to mmSolver, take a look at the video :ref:`tutorial-heading`.
